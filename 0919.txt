2. bower? node?

프런트앤드라이브러리 관리자
npm이라는 모듈에서 굴러간다
npm은 node.js 위에서 굴러간다.

빌드하고 압축할때는 그런트


git bash에서
$ node -v : 설치확인

$ npm install -g bower : bower 설치

$ bower -v : 설치확인

$ bower init : 환경설정

$ bower install jquery --save : 제이쿼리 최신버전 설치

$ bower install bootstrap --save : 부트스트랩 최신버전 설치

중간에 bower_components 지우고
$ bower install : bower 설치
